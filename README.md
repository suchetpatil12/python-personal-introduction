# Personal Introduction Program

## 📌 Project Overview
This is a beginner-level Python project created as part of Week 1 Python Basics learning.

The program asks the user for personal information such as:
- Name
- Age
- Hobby
- City

After collecting the information, the program displays a friendly personalized welcome message.

---

## 🎯 Objectives
The main goals of this project are:
- Learn Python basics
- Understand variables
- Use input() and print() functions
- Practice string formatting using f-strings
- Build confidence in writing simple programs

---

## 🛠️ Technologies Used

## ▶️ How to Run the Project

### Step 1 — Install Python
Download and install Python from:

https://www.python.org/downloads/

---

### Step 2 — Clone or Download the Project

```bash
git clone <your-repository-link>
```

OR download the ZIP file manually.

---

### Step 3 — Open Terminal in Project Folder

Navigate to the project directory:

```bash
cd python-personal-introduction
```

---

### Step 4 — Run the Program

```bash
python personal_intro.py
```

OR

```bash
python3 personal_intro.py
```
